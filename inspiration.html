Getting started with Git & GitHub

To learn how to use both Git and GitHub I watched this video. This was the video that got me to really understand the usage and the way to work with Git and GitHub. It's very informative in an easy way, amongst a lot of things Nick explains the difference between Git and GitHub.

Why it's useful to know:

  1. It enables teams to work together on the same codebase efficiently.
  2. You can track, manage, and revert changes, avoiding mistakes or loss of progress.
  3. GitHub is widely used by developers to showcase their projects, contributing to job opportunities.
  4. Many open-source projects are hosted on GitHub, allowing you to learn and contribute to global projects.
