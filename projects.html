Weather Dashboard with Python and JavaScript

Created a weather dashboard using Python for backend API requests and JavaScript for frontend data visualization. The project fetched real-time weather data from an external API (OpenWeather) and displayed the results dynamically in the browser. CSS was used for styling a responsive layout, and JavaScript enabled features like city search and live updates. Python’s Flask framework handled the server-side logic, communicating weather data to the frontend. This project solidified knowledge in API handling and connecting Python with JavaScript for dynamic applications.

To-Do List with Local Storage

Built a functional to-do list app using HTML, CSS, and JavaScript with Python for the backend API. The app allowed users to create, edit, and delete tasks, with all interactions stored locally using the localStorage API. The app featured a clean UI with modern CSS styling and animations for a smooth user experience. Python was used to implement a backend system for saving tasks remotely, with a RESTful API for task management. The project emphasized front-to-back integration, as well as JavaScript’s interaction with the DOM and browser storage.

Interactive Quiz Game

Developed an interactive quiz game with HTML, CSS, and JavaScript on the frontend, and Python to manage the database of questions and scores. JavaScript dynamically updated the UI as the user progressed through multiple-choice questions. CSS animations were added to highlight correct and incorrect answers, providing immediate feedback. Python's Flask framework was used to serve the questions from a database, and a simple scoreboard feature stored the player’s scores. The game honed skills in creating dynamic web apps, DOM manipulation, and Python server logic.

Personal Budget Tracker

Created a personal budget tracking app using JavaScript for calculations and Python for backend data management. The user could input income, expenses, and track savings goals through an intuitive web interface styled with CSS. JavaScript handled real-time updates to budget charts and visualizations using the Chart.js library, providing a graphical representation of spending habits. Memory Game with JavaScript and Python

Developed a classic memory game using HTML, CSS, and JavaScript for the frontend, with Python for backend score tracking. The game consisted of a grid of cards that users could flip to find matching pairs. JavaScript handled the game logic, including card flipping, matching logic, and the countdown timer. CSS was used to create an attractive, responsive design with smooth card flip animations. Python was integrated to track players' scores and save high scores on the server, allowing players to compete globally.

JavaScript-Powered Snake Game

Created a classic Snake game using HTML, CSS, and JavaScript for browser-based gameplay. JavaScript controlled the game logic, including the snake’s movement, food generation, and collision detection. The game’s layout was crafted with CSS Grid, ensuring a clean and responsive design. This project demonstrated how classic games can be implemented using modern web technologies.
